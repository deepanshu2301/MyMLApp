# MyMLApp-Firebase_MLKit
Android app that provides the implementation of various Google API's.<br />
	API's used->
 		Text Recognition
		Barcode Scanning
		Face Detection
		Image Labeling
		Landmark Detection
		Object Detection
		Language Identification
		Smart Reply
		Translate Text
		Auto ML Vision
# How To Use ?
1> Git clone.
2> Add to your Firebase.
3> Profit !

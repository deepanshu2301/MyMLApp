# MyMLApp-Firebase_MLKit
Android app that provides the implementation of various Google API's.<br />
	API's used-><br />
 		Text Recognition<br />
		Barcode Scanning<br />
		Face Detection<br />
		Image Labeling<br />
		Landmark Detection<br />
		Object Detection<br />
		Language Identification<br />
		Smart Reply<br />
		Translate Text<br />
		Auto ML Vision<br />
# How To Use ?<br />
1> Git clone.<br />
2> Add to your Firebase.<br />
3> Profit !
